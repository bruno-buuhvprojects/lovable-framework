# lovable-framework docs

Developer documentation for **lovable-framework**, built with [VitePress](https://vitepress.dev/). See the main README in the repo root for a short overview.

## Run locally

```bash
npm install
npm run docs:dev
```

Then open `http://localhost:5173` (or the port VitePress prints) in your browser.

## Build

```bash
npm run docs:build
```

Output is in `doc/.vitepress/dist`. Preview with:

```bash
npm run docs:preview
```
